# Health Monitoring System

## 📌 Описание проекта

Health Monitoring System — система мониторинга состояния здоровья с использованием методов машинного обучения и анализа данных.

Проект предназначен для:

* обработки медицинских данных;
* анализа показателей здоровья;
* визуализации данных;
* обучения моделей машинного обучения;
* прогнозирования состояния пациента;
* обнаружения аномалий и рисков.

---

## ⚙️ Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/golub_ns_22/Health_Monitoring_System.git
cd Health_Monitoring_System
```

---

### 2. Создать виртуальное окружение

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / MacOS

```bash
python3 -m venv venv
source venv/bin/activate
```

---

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

---

### 4. Запустить Jupyter Notebook

```bash
jupyter notebook
```

После запуска открыть файл:

```text
Health_Monitoring_System.ipynb
```
